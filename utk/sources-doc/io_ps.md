# File format .ps


## Files

```
src/io/imageIO.hpp  
src/io/imageIO_model.hpp  
src/io/imageIO_ps.hpp
```

## Description


UnicornTK relies on the extension of the input file to determine its type. When the extension is `.ps`, the pointset is drawn into a postscript file.
